# 链表（上）

## 问题引出

- 什么是缓存

  缓存是一种提高数据读取性能的技术，常见的有 CPU 缓存，数据库缓存，浏览器缓存等。

- 缓存淘汰策略

  1. 先进先出 FIFO (First In, First Out)

  2. 最少使用策略 LFU (Least Frequently Used)

  3. 最近最少使用策略 LRU (Least Recently Used)

## 链表 VS 数组

|                |     链表     |      数组      |
| :------------: | :----------: | :------------: |
|    存储结构    | 零散的内存块 | 连续的内存空间 |
|    访问方式    |     指针     |      下标      |
| 随机访问复杂度 |     O(n)     |      O(1)      |
|    插入删除    |     O(1)     |      O(n)      |
